# 💬 Twitter Sentiment Analysis

NLP-based sentiment analysis on Twitter data to classify brand-related tweets as Positive, Negative, Neutral, or Irrelevant.

## 📊 Project Overview

This project analyzes public sentiment from Twitter using Natural Language Processing. It classifies tweets about brands into four sentiment categories and visualizes the distribution using pie charts and bar graphs.

## 📸 Sample Output

> Pie chart showing sentiment distribution:
> - 🟢 Positive: 24.87%
> - 🟠 Negative: 32.47%
> - 🔵 Neutral: 35.42%
> - 🔴 Irrelevant: 7.25%

## 🛠️ Tech Stack

- **Python**
- **Pandas** — data loading and cleaning
- **Matplotlib** — pie chart and bar chart visualization
- **NLP techniques** — text preprocessing

## 📁 Dataset

- Twitter Entity Sentiment Analysis Dataset
- Total tweets analyzed: ~2,304
- Classes: Positive (573), Negative (748), Neutral (816), Irrelevant (167)

## ⚙️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Ankush0004/twitter-sentiment-analysis-nlp.git
cd twitter-sentiment-analysis-nlp
```

2. Install dependencies
```bash
pip install pandas matplotlib
```

3. Run the notebook
```bash
jupyter notebook task4.ipynb
```

## 📌 Key Insights

- Neutral sentiment dominated at 35.4% of all tweets
- Negative sentiment was higher (32.5%) than Positive (24.9%)
- Only 7.25% of tweets were classified as Irrelevant
- Brand perception skews slightly negative on Twitter
